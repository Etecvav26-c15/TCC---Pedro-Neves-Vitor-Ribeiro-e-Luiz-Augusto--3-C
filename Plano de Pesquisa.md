# Plano de Pesquisa

## Título do Projeto:
Sistema Escolar com Banco de Dados em Nuvem para Gerenciamento Acadêmico

## Categoria:
(X) Informática

## Resumo:
Este projeto tem como objetivo desenvolver um sistema escolar utilizando banco de dados em nuvem, com separações simples de matérias visando facilitar o gerenciamento de informações acadêmicas como alunos, professores, notas, planos de aulas, turmas e horários de aulas. Além de um sistema de criptografia com maior proteção de dados para login e senha, uma conexão mais estável com internet para evitar que tenha uma conexão lenta com o Banco de Dados. A pesquisa será baseada no uso de tecnologias modernas como bancos de dados em nuvem e desenvolvimento web. O sistema será desenvolvido utilizando linguagens como PHP, Python ou Java, integradas a um banco de dados online e um sistema de APIs para conexão mobile diretamente ligada ao site Web sem que haja discordâncias entre esses sites. Espera-se como resultado um sistema funcional, acessível e eficiente para uso escolar.

## Palavras-chave:
Banco de dados, Nuvem, Sistema escolar, Desenvolvimento web, Tecnologia

## Introdução:
Com o avanço da tecnologia, o uso de sistemas digitais em escolas se tornou essencial para melhorar a organização e o acesso às informações. Muitas instituições ainda utilizam métodos manuais ou sistemas limitados, o que dificulta o gerenciamento de dados. Além de não utilizarem APIs para um aplicativo Mobile e que seja funcional.

Diante disso, este projeto propõe o desenvolvimento de um sistema escolar integrado a um banco de dados em nuvem, permitindo maior eficiência, segurança e acessibilidade. Com criptografia em Rede, uma conexão mais segura que servidores locais, senha e login criptografados e interconexões entre Web e Mobile.

## Objetivos:

### Objetivo Geral:
Desenvolver um sistema escolar utilizando banco de dados em nuvem para facilitar o gerenciamento de informações acadêmicas.

### Objetivos Específicos:
- Estudar bancos de dados em nuvem  
- Utilizar o phpMyAdmin  
- Desenvolver interface web para o sistema  
- Implementar cadastro de alunos, professores e turmas  
- Criar sistema de notas e controle acadêmico  
- Testar o funcionamento do sistema  
- Utilizar conexões diretas com PHP  
- Utilizar hash de senhas  
- APIs junto ao Mit Inventor  

## Metodologia:
O projeto será desenvolvido por meio de pesquisa bibliográfica e prática. Inicialmente, serão estudadas tecnologias relacionadas a bancos de dados em nuvem, desenvolvimento web, aplicativos mobiles e sistemas de segurança de dados.

Em seguida, será utilizado o phpMyAdmin como banco de dados pela facilidade e boa criptografia de dados.

O sistema será desenvolvido utilizando linguagens de programação como PHP, Python, JavaScript e Java com integração ao banco de dados.

Serão realizados testes para garantir o funcionamento correto do sistema.

## Resultados Esperados:
Espera-se desenvolver um sistema escolar funcional, capaz de armazenar e gerenciar dados de forma eficiente na nuvem e que tenha a funcionalidade idêntica para mobile. O sistema deverá permitir acesso fácil às informações e contribuir para a organização escolar.
